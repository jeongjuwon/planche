== Description ==
Planche is MySQL GUI client tool. It runs on your wordpress. You can enjoy this with opensource on github.
Planche was written in javascript. But this tool will be run like a GUI application.

Support http tunneling.
Support multi connection.
Support multi query editor.
Support multi query result tab.

Just enjoy it!

== Installation ==
1. Upload contents of the directory to /wp-content/plugins/ (or use the automatic installer)
2. Activate the plugin through the \'Plugins\' menu in WordPress
3. Click planche in your admin left sidebar
4. Click \"Run planche\" button
